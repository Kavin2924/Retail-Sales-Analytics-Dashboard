# Retail Sales Analytics Dashboard

## Project Overview
This project analyzes retail sales data using MySQL and Power BI.

## Tools Used
- MySQL
- SQL
- Power BI
- Excel

## Key KPIs
- Total Sales
- Total Orders
- Total Customers
- Average Order Value

## Dashboard Features
- Sales Trend Analysis
- Product Category Analysis
- Customer Analysis
- Gender Analysis
- Top Customers
